搭建一个个人博客需要什么呢？

唯一需要的就是耐心、坚持，你需要在遇到问题时自己去网上查找答案，有时候一个问题就要查半天，最后还有可能没解决，但是当问题解决时的满足感令人愉悦。

1.准备工作

在开始一切之前，你必须已经：

·有一个github账号，没有的话去注册一个（GitHub官网）
·安装了node.js、npm，并了解相关基础知识；
·安装了git for windows（或者其它git客户端）
